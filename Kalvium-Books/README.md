# Kalvium Books
A simple application to manage a catalouge of books. 
Get Demo here :->  https://kalvium-books-s-thakur.netlify.app/

## Tech Stack:
1. React.js
2. Tailwind css
3. React-Forms
4. React-Router-dom
5. Axios library to fetch data
